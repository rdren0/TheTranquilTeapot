COZY CAFE INTERIOR
32×32 Pixel Art Asset Pack

Created by Maygetsu

Thank you for supporting my work!

This pack contains a collection of cozy café-themed pixel art assets designed for RPGs, life simulation games, visual novels, and other 2D projects.

PACKAGE CONTENTS

* 699+ pixel art assets
* Environment tiles (walls, floors, windows, doors)
* Furniture and café equipment
* Decorative props and plants
* Characters and café staff
* Animated cats
* Animated doors
* Steam and shadow effects
* Full spritesheet
* Source Aseprite file

SCALE

This pack uses a 32×32 interior scale for improved detail and readability.

It can be used alongside 16×16 exterior assets by scaling them 2×.
